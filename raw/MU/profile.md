## Business Description

Micron Technology, Inc. designs, develops, manufactures, and sells memory and storage products in the United States, Taiwan, Japan, Mainland China, Hong Kong, Europe, and internationally. The company operates through four business units: Cloud Memory Business Unit; Core Data Center Business Unit; Mobile and Client Business Unit; and Automotive and Embedded Business Unit.

The company provides memory products, including dynamic random access memory (DRAM) components and modules, CXL-based memory, LPDDR components and modules, graphics memory, high-bandwidth memory (HBM), and data center memory products; multichip packages (MCP) comprising embedded multimedia card-based, universal flash storage-based, and NAND-based MCPs; and technology leadership products that include 1y DRAM and G9 NAND technologies.

It also offers storage products, such as data center solid-state drives (SSD), client SSD storage, auto and industrial SSD storage; managed NAND; NAND flash; NOR flash; and memory cards. The company markets its semiconductor memory and storage products under the Micron and Crucial brands.

## Key Products & Services

- **DRAM** - Dynamic random access memory for various applications
- **High-Bandwidth Memory (HBM)** - Memory for AI accelerators and GPUs
- **CXL Memory** - Compute Express Link memory solutions
- **LPDDR** - Low-power DDR memory for mobile devices
- **Graphics Memory** - GDDR memory for graphics cards
- **Data Center SSDs** - Enterprise solid-state drives
- **Client SSDs** - Consumer and client solid-state drives
- **NAND Flash** - Flash storage solutions
- **NOR Flash** - Flash memory for automotive and industrial
- **Crucial Brand** - Consumer memory and storage products
- **Multichip Packages** - Integrated memory solutions

## Market Position

Micron Technology is one of the world's leading memory and storage solutions providers, ranking as one of the top three DRAM and NAND manufacturers globally. The company is a critical supplier for the AI infrastructure buildout, with its high-bandwidth memory (HBM) products being essential for AI accelerators from NVIDIA and AMD. Micron serves the data center, PC, graphics, networking, automotive, industrial, and consumer embedded markets, as well as the smartphone and other mobile-device markets.
