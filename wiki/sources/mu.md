# Micron Technology, Inc. (MU)

Leading memory and storage solutions company. Key supplier of HBM (High Bandwidth Memory) for AI accelerators.

## Business Segments

- **DRAM**: Dynamic random-access memory for servers, PCs, mobile
- **NAND**: Flash storage for SSDs, mobile devices
- **HBM**: High Bandwidth Memory for AI accelerators

## Key Products

- Server DRAM modules
- High Bandwidth Memory (HBM3E)
- Enterprise SSDs
- Mobile memory (LPDDR5)
- Consumer SSDs (Crucial brand)

## Investment Thesis

**Bull case**: HBM shortage benefits pricing; essential component for AI GPUs; memory cycle upturn; CHIPS Act funding for U.S. fabs.

**Bear case**: Cyclical industry; commodity pricing pressure; capital intensive; competition from Samsung, SK Hynix.
