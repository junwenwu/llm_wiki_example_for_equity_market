# Fundamental Analysis

Investment analysis method that evaluates a security's intrinsic value by examining financial statements, industry conditions, and economic factors.

---

## Core Framework

### Financial Statement Analysis
1. **Income Statement**: Revenue, margins, profitability
2. **Balance Sheet**: Assets, liabilities, equity
3. **Cash Flow Statement**: Operating, investing, financing flows

### Key Metrics Categories

| Category | Metrics |
|----------|---------|
| **Valuation** | P/E, P/S, P/B, EV/EBITDA |
| **Profitability** | Gross margin, Operating margin, ROE, ROA |
| **Growth** | Revenue growth, EPS growth, CAGR |
| **Solvency** | Debt/Equity, Interest coverage |
| **Efficiency** | Asset turnover, Inventory days |

## Valuation Methods

| Method | Description | Best For |
|--------|-------------|----------|
| **P/E Ratio** | Price to earnings | Profitable companies |
| **P/S Ratio** | Price to sales | Growth companies |
| **DCF** | Discounted cash flow | Stable cash generators |
| **Comps** | Peer comparison | Relative valuation |

---

## Sector-Specific Considerations

Different sectors require different metrics focus:

| Sector | Key Metrics |
|--------|-------------|
| **Tech** | Revenue growth, margins, TAM |
| **Financials** | NIM, credit quality, book value |
| **Healthcare** | Pipeline, regulatory approval |
| **REITs** | FFO, dividend yield, occupancy |

---

## Contrast with Technical Analysis

| Fundamental | Technical |
|-------------|-----------|
| "What to buy" | "When to buy" |
| Long-term focus | Short-term focus |
| Financial statements | Price charts |
| Intrinsic value | Price patterns |

Many investors combine both approaches.
